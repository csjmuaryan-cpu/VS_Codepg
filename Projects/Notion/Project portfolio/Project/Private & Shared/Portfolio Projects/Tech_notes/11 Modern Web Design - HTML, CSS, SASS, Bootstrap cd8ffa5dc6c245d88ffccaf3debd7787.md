## ğŸ’… SASS/SCSS

**What is SASS?**

SASS (Syntactically Awesome Style Sheets) is a CSS preprocessor that extends CSS with powerful features.

**Key Features:**

- Variables for reusable values
- Nesting for better organization
- Mixins (reusable code blocks)
- Functions and operations
- Partials and imports
- Inheritance with @extend
- Control directives (@if, @for, @each)
