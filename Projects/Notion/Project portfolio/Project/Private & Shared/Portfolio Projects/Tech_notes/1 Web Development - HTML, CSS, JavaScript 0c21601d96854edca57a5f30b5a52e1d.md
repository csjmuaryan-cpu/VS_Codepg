### HTML (HyperText Markup Language)

**What is HTML?**

HTML is the standard markup language for creating web pages. It describes the structure and content of web documents using elements and tags.

**Key Concepts:**

- Document structure (DOCTYPE, html, head, body)
- Semantic elements (header, nav, main, article, section, footer)
- Forms and input validation
- Multimedia elements (audio, video, canvas)
- Accessibility features (ARIA attributes)

### CSS (Cascading Style Sheets)

**What is CSS?**

CSS is used to style and layout web pages. It controls colors, fonts, spacing, positioning, and responsive design.

**Key Concepts:**

- Selectors and specificity
- Box model (margin, border, padding, content)
- Flexbox and Grid layouts
- Responsive design (media queries)
- Animations and transitions
- CSS Variables (Custom Properties)
- Modern CSS features (Container Queries, Cascade Layers)

### JavaScript

**What is JavaScript?**

JavaScript is a programming language that enables interactive web pages. It's essential for client-side web development and also used in backend (Node.js).

**Key Concepts:**

- Variables, data types, and operators
- Functions and scope
- DOM manipulation
- Event handling
- Async programming (Promises, async/await)
- ES6+ features (arrow functions, destructuring, modules)
- Fetch API and AJAX
- Local Storage and Session Storage
