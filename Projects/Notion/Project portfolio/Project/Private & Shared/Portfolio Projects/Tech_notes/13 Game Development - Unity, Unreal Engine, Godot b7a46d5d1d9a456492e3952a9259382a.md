## ğŸ² Godot

**What is Godot?**

Godot is a free, open-source game engine with its own scripting language (GDScript) similar to Python.

**Key Features:**

- **Node-based Scene System** - Hierarchical design
- **GDScript** - Python-like scripting
- **C# Support** - Alternative to GDScript
- **2D and 3D** - Dedicated 2D engine
- **Animation Tools** - Built-in animation editor
- **Physics Engine** - 2D and 3D physics
- **UI System** - Control nodes for interfaces
- **Cross-platform** - Export to many platforms
- **Lightweight** - Small download size
